# Expense Tracker

A simple web-based expense tracker built using Django.

## Features
- User Login & Signup
- Add, Edit, Delete Expenses
- Category-wise tracking
- Total expense calculation

## Tech Stack
- Python
- Django
- HTML, CSS

## How to Run
1. Clone the repo
2. Run `python manage.py runserver`
3. Open in browser
